# Discord Clone
### A clone project meant to demonstrate CSS skills. The responsive design is exactly the same as web app of discord.

_Features:_

- Real Time Crud App
- User Authentication
- Update Profile Picture
- Infinite Scroll
- Pixel Perfect Clone
- Exact Responsive Design

_Technologies Used:_

- Vite
- Firebase
- Jest/RTL/Vitest
- Typescript
- React Router
- Tailwind/SCSS
